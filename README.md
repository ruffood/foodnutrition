# 食物营养数据分析

#### 项目介绍
基于已知的1284种食物进行营养数据分析


#### 使用说明

1. 基于bootstrap table搭建数据列表页面
2. 查看各食物营养成分梯度分布
3. 小工具频道陆续开发系列数据管理功能


#### 安装说明
1. 下载本项目代码
2. 导入data/sql文件夹里的数据库文件
3. 修改data/config/db.php种的数据库配置即可
